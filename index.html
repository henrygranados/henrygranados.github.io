About

Welcome, I am Nathen Watters, native of northern Colorado. I am an enthusiast of all things technology, but emphatically so when it comes to front end development for web and mobile applications.

But my passion doesn't just lie with your typical front end development. My background in tech. began with building circuits and programming embedded systems. With the advent of Cloud Computing and the Internet of Things the need for fluid interfaces between embedded devices with the web and mobile devices is growing. And that is where I come in.

My current stack consists of HTML, CSS, JavaScript, jQuery, Bootstrap and Django for web development, Android for mobile computing, and Selenium WebDriver for constructing automation tests to run against web applications to ensure the apps work as expected.
